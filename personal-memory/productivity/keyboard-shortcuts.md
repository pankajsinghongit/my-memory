## ⚠️ Important Guidelines while touching this note

**Critical Information for Maintaining This File:**

1. **Application Naming Convention:**
   - Use `GLOBAL` for shortcuts that work system-wide across all applications (e.g., Cmd+Q to quit)
   - Use specific application name for app-specific shortcuts (e.g., "Todoist", "Chrome", "VSCode")
   - Use `macOS` only for system-level shortcuts that don't work in all apps

2. **Table Format:**
   - Always maintain the table structure
   - Keep entries sorted alphabetically by Application name
   - GLOBAL shortcuts should appear first in the table

3. **Shortcut Formatting:**
   - Use backticks for shortcuts: `Cmd + Q`
   - Use plus signs (+) between keys with spaces
   - Use proper key names: Cmd, Option, Shift, Ctrl, Fn

4. **Date Format:**
   - Always add the date when a shortcut is added: YYYY-MM-DD
